# Human Development Index (HDI) Prediction using Machine Learning

## 📌 Project Overview

The **Human Development Index (HDI) Prediction** project uses Machine Learning to predict a country's Human Development Index based on key socio-economic indicators. A Flask web application provides an interactive interface where users can enter input values and receive HDI predictions in real time.

---

## ✨ Features

- Predicts Human Development Index (HDI)
- Data preprocessing and feature selection
- Machine Learning model training
- Flask-based web application
- Simple and user-friendly interface
- Real-time prediction

## 🚀 How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/Lokeswari09/AI-ML-and-GEN-AI-Track-Project-Template.git
```

2. Navigate to the Flask folder:

```bash
cd "5. Project Development Phase/Flask"
```

3. Install the required libraries:

```bash
pip install flask pandas numpy scikit-learn
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```text
http://127.0.0.1:5000
```

## 📊 Machine Learning Model

- **Algorithm:** Linear Regression
- **Input Features:**
  - Life Expectancy at Birth
  - Expected Years of Schooling
  - Mean Years of Schooling
  - Gross National Income (GNI) per Capita
- **Output:** Predicted Human Development Index (HDI)
